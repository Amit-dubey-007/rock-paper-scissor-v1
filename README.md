# Rock Paper Scissors Game (V1)

A simple Rock Paper Scissors game built with HTML, CSS and JavaScript.

## Features
- Scoreboard for user and computer
- Random computer choice
- Win / Lose / Draw logic
- Reset button
- Responsive UI

## How to Run
Open `index.html` in your browser.
